Travaux

Application de Recherche dans l'active directory

Le but de l’entreprise avec cette mission était de pouvoir faire des requêtes avec l’Active Directory pour en retirer plusieurs informations (groupes de travail, permissions et informations professionnelles). Technologies et langages concernés : utilisation de Visual Studio 2022, Active Directory, Windows Presentation Foundation et XAML. J’ai tout d’abord commencé par écrire mon code en Python et, dès qu’il a été opérationnel, j’ai pu façonner mon application. L’application possède, pour la recherche utilisateur, une search bar par nom de famille ou prénom. Elle permet donc d’avoir des résultats comme les informations professionnelles (téléphone, e-mail, agence, métier), groupes, sous-groupes et permissions.

Requetes API

Le but de l’entreprise avec cette mission était de pouvoir faire des requêtes avec les API de WEBFLEET et HIBOO pour retirer plusieurs informations liées aux différents véhicules en circulation (position, vitesse, etc.). Technologies et langages concernés : utilisation de Visual Studio 2022, des API WEBFLEET et HIBOO, Excel et une base de données. J’ai tout d’abord commencé par faire mon code en Python dans VS Code, pour le terminer ensuite en PowerShell. Les API sont fonctionnelles, avec extraction des données vers une base de données. Mon code permet donc de faire des requêtes aux API WEBFLEET et HIBOO, puis d’envoyer les données vers une base de données afin que la personne chargée de les analyser puisse le faire plus aisément et rapidement.
